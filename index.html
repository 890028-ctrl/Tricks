CSS Box Model

All CSS elements come with a certain margin and padding. Every element has a different margin and padding. You must eliminate the default margin and padding. To neutralize, do this code, but make sure this comes after any imported code line:

*{ margin: 0; padding: 0; box-sizing: border-box; }
